# 贡献流程

1. Fork 仓库
2. 在 `src/wiki/<分类>/xxx.md` 新建文件
3. 提交 PR

## 注意事项

- 贡献者头像墙（`README.md` / `docs/README_zh-cn.md` / `src/index.md`）中 `<!-- CONTRIBUTORS START/END -->` 之间的内容由 GitHub Action 自动生成，**不要手动修改**，否则下次 Action 跑时会被覆盖
- 头像是 GitHub 自动拉取的，无需手动同步
